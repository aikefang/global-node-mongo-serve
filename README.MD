```

```

### http://localhost:2020/

# Rrror status
* 200 成功
* 200000 用户不存在
* 200001 用户未登录
* 200002 用户账号或密码错误
* 200003 数据已存在

* 20000X 该用户没有权限
* 500000 系统错误
* 500001 未知错误
* 500002 缺少参数
* 500003 参数错误
* 300001 数量超出限制
